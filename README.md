# Monitor de Clima

Aplicação em Streamlit que consulta o clima atual e a previsão dos próximos dias para qualquer cidade, usando a API pública e gratuita Open-Meteo (sem necessidade de chave de API).

## Como executar

```bash
pip install streamlit requests
streamlit run app.py
```

## Tecnologias

- Streamlit
- Requests
- Open-Meteo API
